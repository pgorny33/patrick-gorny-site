# Patrick Gorny Real Estate Website

This is a Vercel-ready Next.js website package.

## Quick Vercel deployment

1. Go to Vercel.com and log in.
2. Click **Add New... → Project**.
3. Choose **Import Third-Party Git Repository → Upload** if available, or create a GitHub repository and upload these files.
4. Deploy the project.
5. In the project dashboard, go to **Settings → Domains**.
6. Add `PatrickGorny.com`.
7. Add `PGResidential.com` and set it to redirect to `PatrickGorny.com`.
